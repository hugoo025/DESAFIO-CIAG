Projeto web que integra a API pública TheMealDB para permitir aos usuários buscar receitas, favoritar pratos e criar receitas locais. O sistema combina consumo de API com armazenamento local, oferecendo uma experiência interativa e organizada.

Tecnologias utilizadas: HTML, CSS e JavaScript.

Como rodar: Instale os 3 códigos que estão no repositório (index.html, styles.css e app.js), após isso, coloque os 3 juntos em uma pasta e abra o arquivo index.html e selecione para abrir o arquivo no navegador de sua preferência. Dessa maneira o site vai estar pronto para ser usado.
Ou utilize esse link para entrar direto no site com Netlify: https://gerenciador-de-receitas.netlify.app/
